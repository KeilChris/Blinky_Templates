# Solution templates pack

This CMSIS-Pack contains CMSIS solution-based reference applications that can be used to verify tools and hardware
setup. The reference applications are:

- [Blank](../Templates/Blank/Blank.csolution.yml), a blank example for development boards with a minimal `mani.c` file.
- [Blinky (Button/LED)](../Templates/Blinky_ButtonLED/Blinky.csolution.yml), a blinky example for development boards
  with an LED and a user button.
- [Blinky (Two LEDs)](../Templates/Blinky_TwoLEDs/Blinky.csolution.yml), a blinky example for development boards with
  two LEDs but no user button.
